## 画面詳細仕様

### 1. 収支ページ（AccountsPage） `/`

**目的**: 家計の収支推移と支出内訳を可視化するホーム画面

#### レイアウト構成
```
┌─────────────────────────────────┐
│        ヘッダー                  │
├─────────────────────────────────┤
│  期間選択: [3ヶ月][6ヶ月][1年]    │
│  グラフ種別: [収支][支出][収入]    │
├─────────────────────────────────┤
│                                  │
│   収支推移グラフ（棒グラフ）      │
│   ← → で月別に推移表示           │
│                                  │
├─────────────────────────────────┤
│  月選択:  ◀ 2024年3月 ▶         │
│  グループ: [カテゴリ][支払元]    │
│  [引落口座]                      │
├─────────────────────────────────┤
│                                  │
│  支出内訳（ドーナツ円グラフ）    │
│  中央: 月合計金額表示            │
│                                  │
├─────────────────────────────────┤
│  凡例リスト:                     │
│  □ 食費       35% ¥52,500       │
│  □ 交通費     20% ¥30,000       │
│  □ 光熱費     15% ¥22,500       │
│  ...                             │
└─────────────────────────────────┘
```

#### 主な機能
| 機能 | 説明 |
|-----|------|
| **期間選択** | 3ヶ月 / 半年 / 1年の収支推移を表示切り替え |
| **グラフ種別** | 収支（正負を色分け）/ 支出（赤）/ 収入（緑） |
| **月ナビゲーション** | ◀/▶ で前月/次月の内訳を切り替え |
| **グループ切替** | カテゴリ別 → 支払元別 → 引落口座別 で集計方法を変更 |
| **プログレスバー** | 各項目の割合を視覚的に表示 |
| **グラフクリック** | 該当する取引の詳細モーダルを開く（実装予定） |

#### 色の使い分け
- **正の収支**: 青（#3b82f6）
- **負の収支**: 赤（#ef4444）
- **支出**: 赤（#ef4444）
- **収入**: 緑（#22c55e）
- **グリッド**: グレー透過（rgba(107,114,128,0.2)）

#### 使用ライブラリ
- **グラフ**: Recharts（BarChart, PieChart）
- **日付操作**: date-fns

### 3. 取引追加ページ（AddTransactionPage） `/add-transaction`

**目的**: 日々の取引をすばやく記録するフォーム画面

#### レイアウト構成
```
┌─────────────────────────────────┐
│        ヘッダー（取引追加）       │
├─────────────────────────────────┤
│  [支出] [収入]                  │  ← タブ切替
├─────────────────────────────────┤
│                                  │
│  【クイック追加テンプレート】     │
│                                  │
│  [朝食][昼食][夜食][カフェ]       │
│  [ガソリン][薬][本][服]          │
│  [テンプレート管理...]           │
│                                  │
├─────────────────────────────────┤
│  【入力フォーム】                │
│                                  │
│  金額:                           │
│  [                  ]  ¥        │
│                                  │
│  カテゴリ:                       │
│  ▦▦▦▦▦▦▦▦▦▦▦▦             │
│  (4列グリッド、アイコン付き)     │
│                                  │
│  支払い方法:                     │
│  [現金][給与口座][クレジット]     │
│                                  │
│  日付:                           │
│  [2024年3月5日]                 │
│                                  │
│  メモ:                           │
│  [                             ] │
│                                  │
│  [                 保存          ] │
│                                  │
└─────────────────────────────────┘
```

#### 主な機能
| 機能 | 説明 |
|-----|------|
| **収入/支出タブ** | 入力モードを切り替え |
| **クイック追加** | 事前定義テンプレートでワンタップ入力 |
| **テンプレート管理** | テンプレートの新規追加/編集/削除 |
| **カテゴリグリッド** | 4列グリッド、アイコン表示で選択しやすく |
| **支払い方法選択** | 口座/カード別に選択 |
| **日付選択** | デフォルトは今日、カレンダーピッカーで変更可能 |
| **自動残高更新** | 口座直接・デビットカード → 即時更新、クレジットカード → ペンディング |

#### モーダル
- `QuickAddTemplateModal` — テンプレートの新規追加/編集

#### ビジネスロジック
```
支払い方法により残高更新の処理が異なる:

口座直接支払い (account)
  → 即座に残高から引く

デビットカード (debit_card)
  → 即座に残高から引く + settledAt を現在日時に設定

クレジットカード (credit_card)
  → 残高は変更しない（ペンディング状態）
  → 引落日に自動精算
```

### 4. 履歴ページ（TransactionsPage） `/transactions`

**目的**: 全取引を一元管理し、検索・フィルタで必要な情報にすばやくアクセス

#### レイアウト構成
```

│  検索: [テキスト検索]            │

├─────────────────────────────────┤
│ 【グループ表示】日付別            │
│                                  │
│ 2024年3月5日（今日）             │
│  ├─ 🍔 食費        ¥2,000       │
│  │  メモ: 昼食                   │
│  └─ 💳 給与        ¥250,000     │
│     メモ: 3月分給与              │
│                                  │
│ 2024年3月4日                     │
│  ├─ ⛽ 交通費       ¥5,000      │
│  └─ 💊 医療        ¥3,500      │
│                                  │
│ 【定期取引】                     │
│  📅 毎月5日: ガス代  ¥6,000      │
│  📅 毎月10日: 保険料 ¥8,000      │
│                                  │
├─────────────────────────────────┤
│  グループ切替: 日付別 →          │
│  [日付別][カテゴリ別][口座別]     │
│  [支払元別]                      │
│                                  │
└─────────────────────────────────┘
```

#### 主な機能
| 機能 | 説明 |
|-----|------|
| **グループ切替** | 日付別 → カテゴリ別 → 口座別 → 支払元別をトグル |
| **テキスト検索** | メモ・カテゴリ名で検索 |
| **フィルタ** | 日付範囲、取引種別、カテゴリ、口座、支払い手段、未精算のみ |
| **ソート** | 日付順、金額順、カテゴリ順（昇順/降順） |
| **定期取引統合** | 実取引と定期取引の発生予定を同一リスト表示 |
| **取引詳細** | タップで編集モーダルを開く |
| **行スワイプ** | （実装検討）削除機能 |

### モーダル・シート共通仕様

#### バックドロップ
```tsx
<div className="fixed inset-0 bg-black/50 z-[1000]" onClick={onClose} />
```

#### 情報表示シート（読み取り専用）
- 上辺: フラット（角丸なし）
- デスクトップ: `sm:rounded-xl`
- 閉じ方: シート外タッチ ✕ ボタン

- 上辺: `rounded-t-xl`（モバイル）

- 背景スクロール: `useBodyScrollLock(true)` で防止
- 閉じ方: シート外タッチ ✕ ボタン
- フッター: 保存ボタン（`w-full`）のみ
