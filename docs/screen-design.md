## 画面詳細仕様

### 1. 収支ページ

**目的**: 家計の収支推移と支出内訳を可視化するホーム画面

#### レイアウト構成
```
┌─────────────────────────────────┐
│        ヘッダー                  │
├─────────────────────────────────┤
│  期間選択: [3ヶ月][6ヶ月][1年]    │
│  グラフ種別: [収支][支出][収入]    │
├─────────────────────────────────┤
│                                  │
│   収支推移グラフ                  │
│   ← → で月別に推移表示           │
│                                  │
├─────────────────────────────────┤
│  月選択:  ◀ 2024年3月 ▶         │
│  グループ: [カテゴリ][支払元]    │
│  [引落口座]                      │
├─────────────────────────────────┤
│                                  │
│  支出内訳グラフ                  │
│  中央: 月合計金額表示            │
│                                  │
├─────────────────────────────────┤
│  凡例リスト:                     │
│  □ 食費       35% ¥52,500       │
│  □ 交通費     20% ¥30,000       │
│  □ 光熱費     15% ¥22,500       │
│  ...                             │
└─────────────────────────────────┘
```

#### 主な機能
| 機能 | 説明 |
|-----|------|
| **期間選択** | 3ヶ月 / 半年 / 1年の収支推移を表示切り替え |
| **グラフ種別** | 収支 / 支出 / 収入の表示方式を切り替え |
| **月ナビゲーション** | 前月/次月の内訳を切り替え |
| **グループ切替** | カテゴリ別 → 支払元別 → 引落口座別で集計方法を変更 |
| **プログレスバー** | 各項目の割合を視覚的に表示 |

### 3. 取引追加ページ

**目的**: 日々の取引をすばやく記録するフォーム画面

#### レイアウト構成
```
┌─────────────────────────────────┐
│        ヘッダー（取引追加）       │
├─────────────────────────────────┤
│  [支出] [収入]                  │  ← タブ切替
├─────────────────────────────────┤
│                                  │
│  【クイック追加テンプレート】     │
│                                  │
│  [朝食][昼食][夜食][カフェ]       │
│  [ガソリン][薬][本][服]          │
│  [テンプレート管理...]           │
│                                  │
├─────────────────────────────────┤
│  【入力フォーム】                │
│                                  │
│  金額: [                  ] ¥    │
│                                  │
│  カテゴリ:                       │
│  ▦▦▦▦▦▦▦▦▦▦▦▦             │
│  (グリッド表示、複数選択)        │
│                                  │
│  支払い方法:                     │
│  [現金][給与口座][クレジット]     │
│                                  │
│  日付: [2024年3月5日]            │
│                                  │
│  メモ: [                       ] │
│                                  │
│  [                 保存          ] │
│                                  │
└─────────────────────────────────┘
```

#### 主な機能
| 機能 | 説明 |
|-----|------|
| **収入/支出タブ** | 入力モードを切り替え |
| **クイック追加** | テンプレートでワンタップ入力 |
| **テンプレート管理** | テンプレートの新規作成・編集・削除 |
| **カテゴリ選択** | グリッド表示で選択しやすく |
| **支払い方法選択** | 口座/カード別に選択 |
| **日付選択** | デフォルトは今日、変更可能 |

### 4. 履歴ページ

**目的**: 全取引を一元管理し、検索・フィルタで必要な情報にすばやくアクセス

#### レイアウト構成
```
┌─────────────────────────────────┐
│ 【ヘッダー】                     │
│  検索: [テキスト検索]            │
│  [フィルタ] [グループ切替]        │
├─────────────────────────────────┤
│ 【グループ表示】日付別            │
│                                  │
│ 2024年3月5日（今日）             │
│  ├─ 🍔 食費        ¥2,000       │
│  │  メモ: 昼食                   │
│  └─ 💳 給与        ¥250,000     │
│     メモ: 3月分給与              │
│                                  │
│ 2024年3月4日                     │
│  ├─ ⛽ 交通費       ¥5,000      │
│  └─ 💊 医療        ¥3,500      │
│                                  │
│ 【定期取引】                     │
│  📅 毎月5日: ガス代  ¥6,000      │
│  📅 毎月10日: 保険料 ¥8,000      │
│                                  │
├─────────────────────────────────┤
│  グループ切替:                   │
│  [日付別][カテゴリ別][口座別]     │
│  [支払元別]                      │
│                                  │
└─────────────────────────────────┘
```

#### 主な機能
| 機能 | 説明 |
|-----|------|
| **グループ切替** | 日付別 / カテゴリ別 / 口座別 / 支払元別の表示切り替え |
| **テキスト検索** | メモ・カテゴリ名で取引を検索 |
| **フィルタ機能** | 日付範囲、取引種別、カテゴリ、口座、支払い手段、未精算で絞り込み |
| **ソート** | 日付、金額、カテゴリで並び替え |
| **定期取引表示** | 実取引と定期取引の予定を統合表示 |
| **取引編集** | タップで取引情報の編集・削除 |

### 5. 設定ページ

**目的**: マスタデータ（メンバー、カテゴリ、カード、定期取引、貯金目標）とアプリ全体の設定を管理

#### レイアウト構成
```
┌─────────────────────────────────┐
│        ヘッダー（設定）           │
├─────────────────────────────────┤
│                                  │
│ ▼ メンバー管理                   │
│   👨 太郎（夫）                  │
│   👩 花子（妻）                  │
│   👶 次郎（子ども）              │
│   [+ 追加]                       │
│                                  │
│ ▼ カテゴリ管理 [支出] [収入]     │
│   🍔 食費                        │
│   ⛽ 交通費                      │
│   💡 光熱費                      │
│   ... (各11項目)                 │
│   [+ 追加]                       │
│                                  │
│ ▼ 支払い手段管理                 │
│   💳 JCBカード（クレジット）     │
│   💳 三井住友デビット（デビット）│
│   [+ 追加]                       │
│                                  │
│ ▼ 定期取引管理                   │
│   [支出を表示] [収入を表示]      │
│   毎月5日: ガス代    ¥6,000      │
│   毎月10日: 保険料   ¥8,000      │
│   [+ 追加]                       │
│                                  │
│ ▼ 貯金目標管理                   │
│   🏠 家の修繕費     ¥1,000,000   │
│   🚗 車購入資金     ¥1,000,000   │
│   [+ 追加]                       │
│                                  │
│ ▼ データ管理                      │
│   [JSON エクスポート]            │
│   [JSON インポート]              │
│   [全データ初期化]                │
│                                  │
└─────────────────────────────────┘
```

#### 主な機能
| セクション | 機能 |
|-----------|-----|
| **メンバー** | 家族メンバーの追加・編集・削除 |
| **カテゴリ** | 支出/収入別にカテゴリを管理、追加・編集・削除 |
| **支払い手段** | クレジットカード・デビットカードの管理、追加・編集・削除 |
| **定期取引** | 毎月/毎日の自動計上設定、編集・削除 |
| **貯金目標** | 目標金額の設定・編集・削除、月別進捗管理 |
| **データ管理** | JSON形式でのエクスポート/インポート、全初期化 |
